# MatrixOps  
Matrix Class in C++ to be used for numerical methods

Definition of class and some basic functions  

Implemented:

  define square matrix (rows)  
  print as per in memory (1D)  
  print visual 2D  
  A(i,j) operator  
  A=B operator  

Yet to Implement:
